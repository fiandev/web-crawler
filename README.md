# web crawler
a super simple and lightweight software to crawler any website

## features
* use proxies to avoid blocking

## installation
```bash
git clone https://github.com/fiandev/web-crawler;
cd web-crawler;
pip install -r requirements.txt;
```

## How To Usage
```bash
python3 app.py <url>
```
